Practice Log

QR-Based Instrument Practice Tracker

A kiosk-friendly web app that tracks music practice sessions through QR code scan-in / scan-out and turns them into live stats, leaderboards, and admin insights.

Manual practice tracking is easy to forget and hard to review. Practice Log solves this by letting students scan a QR code to start/end practice while the system automatically records, validates, and summarizes each session.

Tools Used: React, Vite, Capacitor, Node.js, Express, Supabase, Vercel, Tailwind CSS, Radix UI, jsQR, Cursor

  • Touch-friendly kiosk interface for tablet use
  • QR scan flow: scan once to start, scan again to end
  • Admin dashboard for daily logs, summaries, open sessions, and flagged items
  • Manual correction tools for adding, editing, approving, or deleting sessions
  • Gamified leaderboard with progress bars, grades, and achievement badges
  • Auto rules for duplicate scans, stale sessions, and suspicious short sessions
